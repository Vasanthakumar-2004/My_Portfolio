# My Portfolio

Personal portfolio website built with HTML, CSS, and JavaScript.

## Live Website

[View Portfolio](https://vasanthakumar-2004.github.io/My_Portfolio/)

## Pages

- `index.html` - About, skills, education, projects, contact
- `skill.html` - Skills page
- `education.html` - Education and internship timeline
- `project.html` - Projects showcase
- `Contact.html` - Contact page

## Contact Form

The contact form submits `name`, `email`, and `message` to Google Sheets through a deployed Google Apps Script Web App endpoint.

## Tech Stack

- HTML5
- CSS3
- JavaScript
